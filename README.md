## **Project**  

### Jarvis - Voice Assistant
A Python-based voice assistant that can recognize speech commands and perform actions like opening websites.

**Features:**
- Speech recognition using Google Speech Recognition
- Text-to-speech capabilities
- Voice-activated web commands (YouTube, Google)
- Ambient noise adjustment for better accuracy

**Technologies Used:**
- Python
- speech_recognition library
- pyttsx3 (text-to-speech)
- webbrowser module

**Repository:** [github.com/Faujanshaikh/Jarvis](https://github.com/Faujanshaikh/Jarvis)
